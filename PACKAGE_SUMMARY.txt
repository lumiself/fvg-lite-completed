SYSTEM REQUIREMENTS:
-------------------
Python: 3.7 or higher
Browser: Any modern web browser
OS: Windows 10+, macOS 10.14+, Linux
Network: Localhost access required

INSTALLATION STEPS:
------------------
1. Extract package to desired location
2. Install Python 3.7+ if not already installed
3. Run verification: python verify_installation.py
4. Start system: run start.bat (Windows) or ./start.sh (Linux/Mac)
5. Open frontend/index.html in browser
6. Click "Connect" to establish WebSocket connection

TROUBLESHOOTING:
---------------
- Use frontend/test-websocket.html for connection testing
- Check terminal for server logs
- Verify Python dependencies with verify_installation.py
- Ensure port 8765 is available
